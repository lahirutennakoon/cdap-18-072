Project Charter Document Submission

Project Charter Document was submitted on 23rd February 2018. Project charter document is a project registration form which allows final year students of the B.Sc. (Hon) degree program to enlist in the final year project group. This document includes project title, details of four members in the group, the internal supervisor and the external supervisor, a brief description of the project, workload allocation among four members, scope of the project and main potential outcome of the project. The description of the project entered on this form is not considered as the formal project proposal.
